# To-Do List App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.

## Live Demo
👉 [Click here to try the To-Do List App](https://saif-hossam.github.io/to-do-list-app/)

## Features
- Add new tasks easily
- Delete tasks when completed
- Track progress of tasks (completed, pending, total)
- Real-time updates without page refresh

## Purpose
This project was built to practice DOM manipulation and improve frontend development skills using vanilla JavaScript.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
